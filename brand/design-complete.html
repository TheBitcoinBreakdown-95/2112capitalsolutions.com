MODULE 06 — CODEINLINE / BLOCK / TERMINAL

Code surfaces. Skippable, self-contained.

Code blocks are progressive disclosure — the prose carries the meaning, the block carries the proof. Used sparingly on Bitcoin / AI / self-hosting service pages.

Inline code

After hardware arrival, generate the seed via the device's onboard RNG. Verify entropy by running bitcoin-cli getrandom and cross-referencing the device output. Never type the seed into a connected machine.

Block code

BASH# Verify the SHA-256 of the downloaded image before flashing
sha256sum umbrel-os-1.5.2.img.xz
# Expected: c3a7e5d2... (compare against published value)

# Flash to SD card (replace /dev/sdX with your device)
xz -d umbrel-os-1.5.2.img.xz
sudo dd if=umbrel-os-1.5.2.img of=/dev/sdX bs=4M status=progress conv=fsync
